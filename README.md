# Adjustly

**Track purchases. Catch price drops. Claim adjustments.**

A mobile-first PWA for tracking warehouse purchases and identifying price adjustment opportunities.

## Features

- Manual item entry with receipt image upload
- Price drop detection via promo matching and shelf price verification
- Refund Pass with copy-to-clipboard details
- Local-only storage (no account required)
- Dark mode support

## Tech Stack

Next.js 16 路 TypeScript 路 Tailwind CSS 路 shadcn/ui 路 Zustand

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
